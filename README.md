프로젝트 기능
-----------------

* <h3>공통 기능</h3>
 
  * 로그인 화면
  * 메인 페이지

* <h3>세부 기능</h3>

  * 공지 메일     
   
  * 일정 목록          
  * 주소록     
  * 메시지          
     * 발신함         
     * 수신함        
  * 전자 결재     
    * 기안서 작성    
    * 내 결재 관리      
  * 근태 관리    
     * 출퇴근 목록      
     * 휴가 목록      
     * 출퇴근 관리(관리자만)    
  * 게시판 (댓글기능, 이전-다음글, 조회수 기능, 목록 댓글 개수 표시)    
    * 전체 게시판          
     * 공지 게시판      
     * 질문 게시판        
     * 자유 게시판        
     * 내 작성글    
  * 마이페이지     
    * 나의 정보     
    * 나의 급여      
    * 관리자 페이지(관리자만)
